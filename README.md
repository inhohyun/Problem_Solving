### 코딩테스트 언어
 - 현재는 파이썬으로 진행(공부시간 및 구현시간의 단축을 위함, 스터디 구하기도 쉬움, 싸피 코딩테스트 연습)
 - 코틀린으로 시험을 봐야될시에 그때가서 학습해도 늦지 않음(아직 직무를 정확히 정하지 않았기 때문)
